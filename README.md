# test

A Bun + TypeScript project.

## Getting Started

### Install dependencies:

```bash
bun install
```

### Development

Run the development server:

```bash
bun run dev
```

### Build

Build the project:

```bash
bun run build
```

### Run

Run the main file:

```bash
bun run index.ts
```

## About

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
